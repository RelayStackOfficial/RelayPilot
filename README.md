# RelayPilot

An intelligent dashboard for managing AI relay accounts with comprehensive features for usage tracking, cost management, and account optimization.

## 📋 Features

- **Usage Monitoring** - Real-time tracking of your AI relay account usage
- **Cost Management** - Detailed cost analysis and expense tracking
- **Balance Overview** - View and manage your account balance
- **Check-ins** - Track account check-ins and activity history
- **Rewards System** - Monitor and collect rewards
- **Plugin Management** - Install and manage plugins for extended functionality
- **Dashboard Analytics** - Comprehensive analytics and insights

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/allenjxd/RelayPilot.git

# Navigate to the project directory
cd RelayPilot

# Install dependencies
npm install
# or
yarn install
```

### Usage

```bash
# Start the development server
npm start
# or
yarn start
```

The application will be available at `http://localhost:3000`

## 🔧 Configuration

Configure your AI relay account settings:

1. Create a `.env` file in the root directory
2. Add your configuration variables:
   ```
   REACT_APP_API_URL=your_api_endpoint
   REACT_APP_API_KEY=your_api_key
   ```
