# RelayPilot

[English](./README.md) | 中文文档

一个用于管理 AI 中继账户的智能仪表板，具有使用情况跟踪、成本管理和账户优化的全面功能。

## 📋 功能特性

- **使用情况监控** - 实时跟踪您的 AI 中继账户使用情况
- **成本管理** - 详细的成本分析和支出跟踪
- **余额概览** - 查看和管理您的账户余额
- **打卡签到** - 跟踪账户打卡和活动历史
- **奖励系统** - 监控和收集奖励
- **插件管理** - 安装和管理扩展功能的插件
- **仪表板分析** - 全面的分析和洞察

## 🚀 快速开始

### 前置要求

- Node.js（v14 或更高版本）
- npm 或 yarn 包管理器

### 安装

```bash
# 克隆仓库
git clone https://github.com/allenjxd/RelayPilot.git

# 进入项目目录
cd RelayPilot

# 安装依赖
npm install
# 或
yarn install
```

### 使用

```bash
# 启动开发服务器
npm start
# 或
yarn start
```

应用程序将在 `http://localhost:3000` 可用

## 🔧 配置

配置您的 AI 中继账户设置：

1. 在根目录创建 `.env` 文件
2. 添加您的配置变量：
   ```
   REACT_APP_API_URL=your_api_endpoint
   REACT_APP_API_KEY=your_api_key
   ```
